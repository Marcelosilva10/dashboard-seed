Dashboard seed!

Esse é um boot para aplicativos Node.js que já faz a conexão utilizando Passport, mongoDB e Express.

ou

O que é o Dashboard Seed?

É uma estrutura de início de projeto para sites que precisam de área de usuário, com login e senha, com diversas autenticações: login em base de dados mongoDB local, Facebook, Google e Twitter.

Quais tecnologias ela usa?

A base do projeto é Node.js, utilizando o Handlebars, Passport.js e Express.js. O frontend base é feito usando Grunt, Sass, Compass e Bower.

Tenho que pagar algo pra usar?

Nope! O projeto é gratuito e aberto no GitHub para coloboração e download. Se você tem interesse em ajudar, dá um fork ou manda uma mensagem pra gente por lá.
